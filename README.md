# SemilleroAnalisisEconometrico

Repositorio de código del **Semillero de Análisis Econométrico** — UNAL Bogotá.

Contiene todos los do-files y scripts para la armonización, anonimización y construcción del panel académico de la Facultad de Ciencias Económicas (FCE) 2009–2025.
